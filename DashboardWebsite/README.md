# DashBoard 

* Dashboard of SmartCommunity Platform. Dashboard connected with mobile application and social media channel to gather people's insights by demographic data and develop actionable insights out of it. 

## Technologies Used
* Python
* Flask
* D3.Js
* Leaflet
* Twitter API
